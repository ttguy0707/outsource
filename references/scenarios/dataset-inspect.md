# 步骤（按数据形态选做，每步留真实输出作证据）
1. 规模与结构：样本总数、train/val/test 各多少、字段/列名与类型（或图像数+分辨率分布）

- 只报事实，不替人决定怎么处理。
- 跑不动（缺库/超内存/格式不支持）如实报，别硬跑。
